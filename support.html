자주 묻는 질문 / FAQ

알림이 오지 않아요 / Notifications aren't working

iOS 설정 → Alarmz → 알림에서 권한이 켜져 있는지 확인해 주세요. 집중 모드(Focus)가 켜져 있어도 알림이 억제될 수 있습니다.
Check Settings → Alarmz → Notifications. Focus mode may also silence alerts.

기록이 사라졌어요 / My session history is gone

앱 데이터는 기기에만 저장됩니다. 앱을 삭제하거나 기기를 초기화하면 기록이 사라집니다. 현재 클라우드 동기화는 지원하지 않습니다.
All data is stored locally. Uninstalling the app removes your history. Cloud sync is not yet supported.

언어를 바꾸고 싶어요 / How do I change the language?

앱 내 설정 다이얼로그에서 언어를 변경할 수 있습니다. 지원 언어: 한국어, 영어, 일본어, 간체/번체 중국어.
Open the settings dialog inside the app. Supported: Korean, English, Japanese, Simplified/Traditional Chinese.

어떤 호흡 패턴을 사용해야 하나요? / Which breathing pattern should I use?

잠들기 전이라면 Deep 4-7-8, 집중이 필요할 땐 Box 4-4-4-4, 하루 종일 편안하게 쓰려면 Coherent 5-5를 추천합니다.
Sleep: Deep 4-7-8. Focus: Box 4-4-4-4. Daily calm: Coherent 5-5.
